Passor para rodar os projetos:

Estando na pasta API-Go-Rest faça:
Passo 1: Iniciar todos os serviços definidos no arquivo docker-compose.yml com o comando "docker compose up"
Passo 2: Iniciar o servidor Rest com Go com o comando "go run main.go"

Estando na pasta Frontend-Consome-API-Go-Rest faça:
Passo 3: Iniciar o front com o comando "npm start"
